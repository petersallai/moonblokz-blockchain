# moonblokz-blockchain

Authoritative MoonBlokz blockchain interpretation crate.

- `no_std`, no-alloc Rust library.
- Owns chain truth: intake, classification, lifecycle, branch selection, chain-switch, transactions, balances, creator selection, bounded retention.
- Designed for RP2040-LoRa-class microcontrollers; the same crate build runs on the std host simulator.
- Scaffold; story-by-story implementation tracked in `_bmad-output/implementation-artifacts/sprint-status.yaml`.
